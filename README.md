# GENESIS Deliverables

本目录放本轮新生成的讲解与汇报材料。

## 按读者与内容的阅读导航

| 文档 | 对象 | 预计时长 | 前置 |
|---|---|---|---|
| [genesis_explain_high_school.md](./genesis_explain_high_school.md) | 高中生 / 完全无基础 | ~25 min | 无 |
| [genesis_explain_undergrad.md](./genesis_explain_undergrad.md) | 任何工程/理科本科生 | ~45 min | 高中数学，知道神经网络是什么就行 |
| [genesis_explain_grad.md](./genesis_explain_grad.md) | 研究生 / 可讨论方向 | ~90 min | 最优化、ODE、概率论基础 |
| [对比实验_大学生版.md](./对比实验_大学生版.md) | 任何工程/理科本科生 | ~45 min | 高中数学，知道神经网络是什么就行 |
| [genesis_teacher_report_doc.md](./genesis_teacher_report_doc.md) | 汇报场景 | ~30 min | — |
| [genesis_teacher_report_ppt_outline.md](./genesis_teacher_report_ppt_outline.md) | 汇报场景 | ~20 min | — |
| `genesis_teacher_report.pptx` | 可直接打开的 PPT | — | — |

## 各文档一句话定位

- `genesis_explain_high_school.md` — 零公式纯直觉
- `genesis_explain_undergrad.md` — 公式 + 原理推导
- `genesis_explain_grad.md` — 完整数学 + 边界 + 未解决问题
- `对比实验_大学生版.md` — **专讲"老师反馈之后"做的 7 个补充实验（task-010~017）**，重点数据对比
- `genesis_teacher_report_doc.md` — 汇报逻辑、每部分目的、建议讲法
- `genesis_teacher_report_ppt_outline.md` — PPT 逐页大纲
- `genesis_teacher_report.pptx` — 由 `scripts/build_genesis_teacher_ppt.py` 生成

## 建议阅读顺序

1. 先读 `genesis_explain_high_school.md`（建立直觉）
2. 再读 `genesis_explain_undergrad.md`（理解公式与数据）
3. 读 **`对比实验_大学生版.md`**（看老师反馈后新增的三张对比图）
4. 最后读 `genesis_explain_grad.md`（掌握理论细节）
5. 准备汇报时：`genesis_teacher_report_doc.md` → `genesis_teacher_report_ppt_outline.md` → `genesis_teacher_report.pptx`
